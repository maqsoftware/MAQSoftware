Gantt Chart

Monitor key performance indicators beyond project milestones and deadlines with a unique grid feature.



Explore a complete overview of your projects, including individual tasks and their progress. Gantt charts enable users to easily schedule and manage tasks. Ordinary Gantt charts only display basic details such as task IDs, task names, and schedules. Gantt Chart by MAQ Software also includes a grid where users can include additional task information such as budget, project status, and location. With Gantt Chart, you can also group tasks by category and sort for a specific data point.


Try it now

Business Uses

  • Project Managers: Monitor project IDs, start times, end times, status, duration, project owners, task priorities, and other relevant KPIs
  • Sales and Marketing: Schedule a campaign while tracking for budget, team input, and sales metrics
  • Admin: Break down company-wide tasks into manageable sprints that measure for success criteria such as completion date and risk status

Key Features

  • Hierarchical view of grouped tasks with an expand/collapse option for each category
  • Color bars based on task category
  • Configurable color, data point radius, data labels, plot color and transparency, stroke width, legend position, and legend formatting
  • Expandable grid with additional task details
  • Task status in relation to the current date
  • The ability to sort data by any grid detail
  • Tooltips for column headers, category labels, KPI indicators, data labels, and bars
