Forecast Using Neural Network

Forecast future values based on historical data using a Neural Network algorithm.



Leverage machine learning to forecast organizational data. Forecasting using Neural Network by MAQ Software implements an “Artificial Neural Network” to learn from historical data and predict future values. This visual uses a single layer feed forward network with lagged inputs to process time series values.


R package dependencies (auto-installed): forecast, plotly, zoo, xts.


Try it now

Business Uses

  • Budgeting: Forecast future project budget based on data from previous quarters
  • Inventory: Analyze inventory needs based on past stock data
  • Sales: Forecast sales and customer demand based on past data

Key Features

  • Configurable time format (can use years or distinct numerical values in place of date and time field)
  • Functionality for both numerical series (years or numbers) and proper date and time values
  • Hover tooltips and highlighting of specific portions of the plot
  • Capability to manually adjust the learning model parameters
