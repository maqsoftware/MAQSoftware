CertyFAST by MAQ Software

Certify your Power BI reports

5 minutes to read   |   Posted: 27th May 2024

CertyFAST by MAQ Software is a Power BI External Tool designed to improve the development and review process of Power BI models and reports. This tool helps team members to easily identify common mistakes in the Power BI model and assists developers in formatting DAX measures used in the report. Additionally, it aids in documenting the model with a single click, easing the process of identifying the fields used in various visuals.

With the increasing adoption of Power BI and the trend towards self-serve reporting, many organizations are dealing with large enterprise models with multiple tables and measures. This scale can make it challenging for developers or reviewers to perform comprehensive checks for the entire model, potentially leading to sub-optimal deliverables. CertyFAST by MAQ Software addresses this challenge by enabling basic checks with a single click, highlighting gaps in the overall model against industry best practices.

However, as the adoption of Power BI grows, organizations face a range of challenges related to governance, user experience, report performance, brand consistency, use of custom components, and solution architecture. These challenges necessitate tools that can help developers and reviewers ensure quality and adherence to standards in Power BI solutions.

How to Use CertyFAST & its Features

    1. Home Page: Select Semantic Model and Report
        Open Power BI report in your desktop to be reflected in the semantic model dropdown
    2. Insights Page: Identify all the gaps against organizational standards and best practices from a single view.
      1. Navigate between errors and warnings to see the gaps based on severity
      2. Rulesets are categorized into logical section based on the impacted Power BI object (tables, columns, measures, visuals)
      3. Click on any ruleset to view the details of non-compliant object
      4. Click on lineage icon to understand the image of the selected object
    3. Lineage Page: Perform impact analysis for any field
      1. How to ignore/whitelist specific words from Spell Check: In the installation directory (C:\Program Files\CertyFAST), update SpellCheckIgnore.txt with the words to be whitelisted (separated by a new line).

Report UI Analysis

  • Upload .pbix for Report UI Analysis.



  • You are now ready to access the features.
  • Features:
    • Report and Page summary
      • Provides a summary of the pages, visuals, and other visual components present in the report.



      • Spell Check: Highlights possible errors in visual titles, axis titles, button text, etc.



    • Format properties
      • Displays the formatting properties of all visual components present in the report.



      • When a custom theme is applied to the report, all the deviations from the theme are highlighted in the grid.



      • When no custom theme is applied to the report, you can define your own properties under the theme section. Any deviations from the provided values will be highlighted in the grid.



    • Visual Summary
      • Visual Summary displays information about all the visuals present on the selected page.
      • Canvas displays a layout preview of all report pages.



      • Grid displays information about individual visuals like their titles, columns and measures used.



      • Export Layout Summary: Ability to export Report, Page, Spell Check & Visual Summary information to an Excel sheet along with a new sheet containing a list of all the fields which are being used in a specific Power BI report.



Callouts

  • Format Properties page will extract format properties for the default Power BI visuals only (out of the box visuals).
  • Model Analysis is not supported for Live connected reports. The actual dataset file for model analysis needs to be used.
