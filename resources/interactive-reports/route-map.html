Flight Route Map Dashboard

Overview

Showcases the flights from different airlines originating from a source location to a destination location across the globe using an animated map visual. Allows users to filter the results by various airline companies. Highlight the top airlines by operating fleet. Gain insights on flight details with origin, destination, flight name and total travellers aboard. Also showcase the top airlines by total passengers/travellers.

Features:

  • Allow users to view the different airlines originating from a source location to a destination location using an animated map visual
  • Displays a summarised view of total travellers and total flights
  • Shows the top performing airlines by operating fleet
  • Ability to the users to filter the results by various airline companies
  • Shows the top performing airlines by total travellers
  • Provides insights on flight details with origin, destination, flight name and total travellers aboard

UI/UX Solution:

The flight route map visual provides an animated view of the route map of various flights operated by the airline companies originating from a source location to a destination location. The visual is very intuitive and users can easily visually interpret the top performing airline companies, routes in terms of operating fleet & passengers using the secondary visuals.
